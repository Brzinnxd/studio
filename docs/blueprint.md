# **App Name**: Angel's Sweets Emporium

## Core Features:

- Product Catalog: Display a catalog of sweets with detailed descriptions and images. Highlight 'Balas Baianas' as the flagship product.
- Order Placement: Enable customers to place orders for the sweets through the system.
- Order Management: Allow the company to manage and process orders received.
- Inventory Tracking: Track the inventory of each product to ensure availability.
- Customer Profiles: Maintain customer profiles with order history and preferences.
- Trend Analysis: Employ a tool which performs data analysis of successful and less successful product pairings in the sales data and provides information about current and possible trending flavors

## Style Guidelines:

- Primary color: Pastel pink (#FFB6C1) to align with the brand's aesthetic.
- Background color: Very light pink (#FDE2E4) for a soft, minimal backdrop.
- Accent color: Light peach (#FFDAB9) for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a modern yet approachable feel.
- Use simple, custom-designed icons relevant to each sweet or category.
- A clean, minimalist layout to highlight products without clutter.
- Subtle animations and transitions for a smooth user experience.
